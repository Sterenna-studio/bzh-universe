# BZH HUB — BZH PW / BZH POWER / BZH CHRONICLES

**Version de centralisation : 2026-05-20**

Ce dépôt est un hub documentaire complet pour regrouper, organiser et faire évoluer tout ce qui concerne :

- BZH PW
- BZH POWER
- BZH CHRONICLES
- BZH CARD GAME: Power Your Hand
- les déclinaisons musicales, visuelles, merchandising et projets annexes

## Objectifs

1. Éviter la dispersion des idées entre conversations, prompts, prototypes et médias.
2. Poser une bible de référence pour l’univers BZH.
3. Structurer les sous-projets existants :
   - jeu de cartes ;
   - univers fictionnel ;
   - musique / album ;
   - merch ;
   - certificats ;
   - projets techniques / jeux expérimentaux.
4. Préparer un dépôt Git exploitable et extensible.

## Positionnement

Ce hub distingue clairement :

- validé / stable ;
- fortement établi ;
- présent dans les échanges mais à confirmer ;
- piste créative à explorer.

Les médias historiques ne sont pas tous réintégrables automatiquement ici ; le repo contient donc un catalogue dédié et tous les emplacements prêts à les accueillir.
