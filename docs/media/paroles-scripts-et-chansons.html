Paroles, scripts et chansons — état récupéré

Ce document rassemble ce qui est disponible avec suffisamment de précision dans les anciennes conversations.

1. BZH POWER — chanson / anthem

Éléments confirmés

  • Morceau conçu comme un hymne spectaculaire et volontairement excessif.
  • Présence de lignes à voix robotique / vocaloid.
  • Gimmick récurrent :
    • B-B-B-B-B-ZH POWER!
  • Présence d’une formule marquante :
    • GIGA-SUPRA-MEGA-LASERO-DESTROYEURE de monde

Remarque importante

Une version complète structurée a bien été produite dans un ancien échange :

  • intro ;
  • refrains ;
  • couplets ;
  • voix robotique ;
  • effets sonores ;
  • répétitions de slogans.

Le HUB V2 note son existence, ses éléments distinctifs et son rôle. Le texte intégral mot pour mot n’est pas reconstitué ici lorsqu’il n’est pas retrouvé textuellement.

2. L’Heure du Combat

Rôle

Discours / morceau d’introduction avant bataille pour BZH POWER.

Distribution vocale

  • MutenRock — Male1
  • Sniky — Male2
  • Aligax — Woman1

Structure récupérée

  1. Intro radio / interférences
    • ambiance de transmission brouillée ;
    • tension d’avant-combat.
  2. Prise de parole de MutenRock
    • ton de leader ;
    • rassemblement des forces.
  3. Relais de Sniky et Aligax
    • renfort du caractère menaçant et combatif.
  4. Jingle héroïque / cri d’équipe
  5. Pont
  6. Outro

Formulations exactes conservées

  • Nous sommes le bug, nous sommes l’anomalie. Et cette nuit, nous écrivons notre propre programme.
  • BZH POWER ! Nous sommes l’erreur qu’ils n’ont pas prévu !

Usage possible

  • ouverture d’EP ;
  • intro de clip ;
  • trailer de sortie ;
  • scène de ralliement des personnages.

3. Glitched Warcry

Nature

Titre DnB / cri de guerre glitché, pensé pour prolonger l’énergie de L’Heure du Combat.

Caractéristiques retrouvées

  • Intro vocale ;
  • montée agressive ;
  • drop DnB ;
  • répétitions chorales glitchées autour du nom d’équipe ;
  • structure pensée pour faire basculer du discours vers l’action.

Statut

La structure complète a été travaillée auparavant ; le HUB V2 conserve son identité, son usage et sa place dans le corpus.

4. Spirale de Feu

Nature

Morceau explosif de l’univers BZH Chronicles avec :

  • MutenRock ;
  • Aligax ;
  • Sniky.

Direction demandée

  • Hybridation glitch-hop + orchestral cinematic
  • Couleur de bataille dystopique
  • Sensation de montée presque rituelle

Structure évoquée

  • Intro
  • Drop 1
  • Refrain
  • Bridge
  • Drop 2
  • Outro

Fragments de direction retrouvés

  • Sniky : voix murmurée
  • MutenRock : voix grave
  • Aligax : voix froide
  • Drop scandé autour de :
    • B-Z-H!
    • SPIRAAAAALE!
    • DELETE!
    • RESET!
