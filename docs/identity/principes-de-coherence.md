# Principes de cohérence

## À conserver
- mélange organique / technologique ;
- ancrage BZH lisible ;
- densité maîtrisée ;
- symboles structurants.

## À éviter
- cyberpunk générique ;
- surcharge de logos ;
- perte de silhouette ;
- trop d’effets sans hiérarchie.

## Pixel art
Lorsqu’un brief l’exige :
- utiliser strictement la palette Cn ;
- ne pas introduire de couleurs extérieures ;
- quantifier à la couleur la plus proche ;
- pas de dithering par défaut.
