# Direction artistique

## Style global
Fusion :
- cyberpunk sombre ;
- fantastique numérique ;
- ambiance néo-bretonne ;
- univers post-civilisation.

## Palette
- Bleus électriques
- Violets profonds
- Rouges / pourpres saturés
- Ocre métallique / cuivre
- Noirs texturés / gris chromés
- Accents néon verts ou turquoises

## Textures
- Peau synthétique
- Ronces de métal
- Circuits intégrés
- Métal rayé
- Cuir digitalisé
- Béton fracturé
- Verre rétroéclairé
- Biomécanique

## Typographie
- Titres : runique + dystopique
- Secondaire : monospace / OCR / data

## Environnements
- Villes suspendues
- Brume néon
- Vaisseaux cyber-nautiques bretons
- Ruines naturelles + artefacts technos
- HUD translucides et triskels
