# Vision globale

## Les grands ensembles

### BZH PW
**BZH PW** est le noyau identitaire / communautaire / symbolique.  
Il sert de bannière commune à plusieurs créations : visuels, merch, emblèmes, éléments pixel art, objets de collection et créations transverses.

### BZH CHRONICLES
**BZH CHRONICLES** constitue l’axe narratif et fictionnel principal.  
L’univers mélange :
- cyberpunk sombre ;
- fantastique numérique ;
- ambiance néo-bretonne ;
- ruines post-civilisationnelles ;
- symbolisme celtique modernisé ;
- technologies usées, organiques, presque mystiques.

### Projets dérivés
Le socle BZH PW / BZH CHRONICLES alimente plusieurs formats :
- **BZH CARD GAME: Power Your Hand** ;
- visuels de merch et objets physiques ;
- créations d’albums / pistes musicales ;
- certificats, diplômes, affiches, logos, wallpapers ;
- éléments pixel art et variantes graphiques.

## Intention créative

Le cœur de l’univers repose sur un contraste :
- **mémoire, fraternité, traces d’un groupe** ;
- **monde fracturé par le Code** ;
- **technologie, pouvoir, reliques et nature** ;
- **équipe de personnages plus grands que nature**, entre légende de crew et mythologie personnelle.

## Principes à conserver
- Ne pas lisser l’identité en une franchise générique.
- Garder la dimension très personnelle, presque “légende de bande”.
- Autoriser plusieurs tonalités : épique, absurde, artefact, collector, jeu.
- Centraliser les décisions ici dès qu’elles deviennent réutilisables.
