# Méthodologie de citations

## Objectif
Préserver les formulations originales de Pierre autour de :
- BZH PW
- BZH POWER
- BZH CHRONICLES

afin que le repo ne soit pas seulement une synthèse éditoriale, mais aussi une **archive de référence**.

## Trois niveaux de traçabilité

### 1. Verbatim retrouvé
Le texte est reproduit tel qu’il a été récupéré.

Format :
> « citation exacte »

### 2. Fragment exact
Une partie du message est disponible de manière fiable, mais pas nécessairement l’intégralité du message original.

Format :
> Fragment exact retrouvé : « ... »

### 3. Synthèse non verbatim
Le sens de la demande est connu, mais pas la formulation complète.
Dans ce cas, le HUB **ne simule pas une citation** : il indique simplement le contenu synthétisé.

## Règle d’intégrité
Aucune formulation ne doit être présentée comme une citation originale si elle n’a pas été retrouvée de manière suffisamment fiable.
