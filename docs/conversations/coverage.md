# Couverture de l’archive de conversations

## Volume
- Entrées archivées : **45**
- Période couverte : **2 avril 2024 → 20 mai 2026**

## Catégories couvertes
- musique et EP ;
- lore et personnages ;
- card game / BZH01 ;
- pages web et mini-sites ;
- trailers et communication ;
- merch, lampes, posters, objets ;
- classement Steam ;
- roguelite ;
- hub Minitel / PC HUB 3D ;
- centralisation Git / HUB.

## Limite honnête
Cette V4 est la passe la plus complète réalisée ici, mais elle ne garantit pas encore :
- une transcription intégrale mot à mot de toutes les anciennes conversations ;
- la récupération binaire automatique de tous les anciens ZIP, PDFs, SVG ou médias ;
- la restitution de tous les messages assistants historiques dans leur texte complet.

Elle conserve toutefois :
- les messages originaux retrouvés ;
- les décisions ;
- les livrables ;
- les contenus de conversation structurants ;
- la chronologie du projet.
