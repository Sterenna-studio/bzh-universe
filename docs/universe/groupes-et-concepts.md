# Groupes et concepts

## BZH Chronicles
Le nom désigne autant :
- l’univers ;
- le collectif de figures qui le portent ;
- un ensemble de récits, objets, images et légendes internes.

## Attestation d’Ancien
Un format de certificat / artefact visuel a été validé autour de la reconnaissance d’un “Ancien des BZH Chronicles”.

Exemple déjà établi :
- **Titre** : `ANCIEN — CONRAD_OG`
- **Formulation centrale** :  
  *Découvert avant la stabilisation, actif durant les fractures, toujours présent après les resets.*
- **Bloc profil** :
  - Apprenti sorcier à l’Université des Lemovices
  - Désosseur de chrome à temps partiel
  - La colline a des yeux 👁️

Contraintes de mise en page évoquées :
- structure proche du modèle original ;
- ajout d’un triskel à gauche ;
- petit fantôme cochon d’Inde “Baba” en haut.

## Concept à développer : Le Code
Dans la direction artistique, le monde est “fracturé par le Code”.  
Ce terme peut devenir :
- un phénomène cosmique ;
- une corruption de réalité ;
- une force politique ;
- une couche techno-mystique ;
- un fil rouge du lore.

> À arbitrer dans une future passe de worldbuilding.
