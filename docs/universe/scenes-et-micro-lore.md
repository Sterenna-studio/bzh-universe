# Scènes, temple et micro-lore

## 1. Le temple
Éléments retrouvés dans les anciens échanges :
- progression à travers un temple ;
- gardiens par étage ;
- MutenRock comme meneur de l’avancée ;
- Sniky opérant aussi par délégation extérieure via son lieutenant ;
- possibilité d’une montée dramatique sur plusieurs niveaux.

## 2. Aligax et MutenRock
Une variante forte de lore établit qu’Aligax est transformée en armure cybernétique autour de MutenRock, créant une entité fusionnée.

Statut :
- important ;
- encore à décider comme canon principal ou version alternative.

## 3. Roster narratif étendu

## 4. Ton recherché
- légende de groupe ;

- capacité à basculer dans l’épique pur.
