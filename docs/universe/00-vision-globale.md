# Vision globale

BZH CHRONICLES transforme une identité de groupe en mythologie cyber-fantasy.

L’univers repose sur :
- mémoire ;
- puissance ;
- fraternité ;
- technologie ;
- ruines ;
- héritage symbolique ;
- corruption ou fracture du réel par le Code.

Ce cadre doit pouvoir accueillir :
- des personnages ;
- des musiques ;
- des objets ;
- des jeux ;
- des certificats ;
- du merch ;
- des récits plus longs.
