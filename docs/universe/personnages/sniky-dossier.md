# Sniky — dossier détaillé

## Rôle
- Assassin furtif
- Allégeance : BZH POWER
- Figure de l’ombre et de l’exécution silencieuse

## Mission spécifique retrouvée
Sniky possède un **lieutenant loyal** chargé de neutraliser ou gérer les cibles **à l’extérieur du temple** pendant que l’équipe progresse à l’intérieur.

## Description physique et équipement
Éléments retrouvés :
- habits sombres ;
- masque avec vision thermique / nocturne ;
- tenue tactique furtive ;
- lames énergétiques ;
- drones ;
- brouilleurs.

## Compétences
- infiltration ;
- combat furtif ;
- renseignement ;
- collecte discrète d’informations ;
- élimination ciblée.

## Origines
Une fiche ancienne le décrivait comme issu des **bas-fonds cyberpunk**, recruté par MutenRock.

## Relations
- **MutenRock** : respect mutuel, lien d’efficacité.
- **Aligax** : méfiance / tension potentielle.
- **Spirit** : collaboration tacite.
- **Dr. Sorn** : réserve ou méfiance.
- **Gabilone** : complémentarité opérationnelle.
