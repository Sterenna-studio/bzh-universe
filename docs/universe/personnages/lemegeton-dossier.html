LEMEGETON - dossier personnage

Statut

  • Type : robot Minitel / personnage-interface / asset technique.
  • Statut narratif : reference avec canon a confirmer.
  • Statut production : pack sprite exploitable, classe et consultable.
  • Entree pivot : Personnages.

Role narratif

LEMEGETON est actuellement le point de jonction entre personnage, interface et archive Minitel. Le lot classe le montre comme une presence ecran animee : corps de terminal, masques, emotions, marche, idle et variantes chroma.

La piste la plus solide est de le traiter comme une figure d'interface expressive pour les surfaces BZH Chronicles FM, Minitel HUB 3D ou MINI-STAR, plutot que comme un protagoniste complet tant que le canon n'a pas ete arbitre.
