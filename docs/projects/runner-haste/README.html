Prototype runner BZH / adaptation Haste

Statut dans le hub

  • Type : prototype jeu / game design.
  • Statut : piste, base de design a consolider.
  • Entree pivot : Carte des projets.

Concept retrouvé

Projet de jeu 2D side-scroller auto-run inspiré de Haste.

Principes clés

  • jump ;
  • dash / boost ;
  • momentum ;
  • système de perfect landing ;
  • terrain procédural ;
  • menace qui poursuit le joueur :
    • Vague du Code.

Mode visualizer prévu

  • Web Audio FFT pour piloter :
    • la vague ;
    • le terrain ;
    • certains rythmes visuels.
  • Source locale recommandée au départ ;
  • URL ensuite, sous réserve de problèmes CORS.

Intention

Adapter l’énergie de Haste à l’univers BZH :

  • vitesse ;
  • pression permanente ;
  • montée sonore / visuelle ;
  • menace arrière symbolisant le Code.
