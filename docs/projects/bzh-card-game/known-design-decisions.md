# Décisions connues — BZH CARD GAME

## Cadre rapide
Le jeu vise la lisibilité et la rapidité :
- moins de surcharge ;
- decks courts ;
- combats qui se règlent vite ;
- identité de Champion forte.

## Décisions déjà posées
| Élément | Décision |
|---|---|
| HP de départ | 15 |
| Taille du deck | 20 cartes |
| Champions | 1 par deck |
| Mana | +1 par tour |
| Affinités | Non |
| Face avant | Nom + type uniquement |
| Set initial | BZH01 |

## Point important
Certaines discussions anciennes contenaient des valeurs antérieures plus élevées pour les HP ou la taille du deck.  
La version à conserver ici est celle déjà stabilisée :
- **15 HP** ;
- **20 cartes**.
