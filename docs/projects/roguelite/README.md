## Détails retrouvés — v3.1
Livraison historique :
- `bzh-chronicles-roguelite-v3_1.zip`
- `index.html`
- `game.js`

### Changements signalés
- recoil léger configurable ;
- micro hitstop ;
- screen shake subtil ;
- stats visibles en pause ;
- affichage des upgrades sélectionnés sur les côtés ;
- mémorisation des **16 derniers upgrades** visibles ;
- shards confinés à l’arène.

## Détails retrouvés — v3.2
Une itération suivante est mentionnée :
- recoil de base **0.20** ;
- `+0.05` par upgrade ;
- hitstop seulement sur boss / élites (`maxHp > 5`) ;
- barre d’énergie secondaire placée sous `2nd` dans le HUD ;
- icônes BZH dans les panneaux d’upgrade.

## Direction de design
Le projet vise :
- plus de ressenti ;
- meilleure lisibilité ;
- meilleure tension sur les cibles fortes ;
- une identité plus explicitement BZH dans l’UI.
