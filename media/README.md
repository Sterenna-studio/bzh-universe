# Media

Ce dossier accueille les médias réels :
- logos ;
- covers ;
- audio ;
- vidéos ;
- exports de certificats ;
- visuels merch ;
- webtoon / affiches ;
- overlays stream et emotes ;
- icônes sociales et raccourcis ;
- références visuelles à valider.

Les médias historiques ne sont pas tous disponibles dans ce ZIP, mais le catalogue ci-dessous indique quoi réintégrer.

## Audio classe
- `audio/tracks/` — pistes MP3 issues des imports.
- `audio/previews/` — extraits ou musiques de site.
- `audio/masters/` — sources WAV conservees comme masters.
- `audio/tracks/sniky-the-frager-mix/` — variantes MP3 Sniky importees depuis le lot Desktop BZH.
- `audio/tracks/dernier-souffle/` — pistes et variantes MP3 Dernier souffle / Echo Vide / Fantome de Chair.
