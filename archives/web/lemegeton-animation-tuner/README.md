# Lemegeton Animation Tuner

Petite page locale pour tester une sprite sheet et trouver une cadence adaptée.

## Lancement

Ouvrir `index.html` dans un navigateur moderne.

La sprite sheet `assets/walk_leme_2.png` est chargée par défaut. Il est aussi possible de charger une autre image depuis l’interface.

## Fonctions

- nombre de colonnes et lignes ;
- plage de frames ;
- FPS précis et multiplicateur de vitesse ;
- boucle normale, ping-pong ou lecture unique ;
- lecture normale ou inversée ;
- comparaison A/B de deux cadences ;
- fond damier, sombre, clair ou chroma ;
- diagnostic automatique ;
- export JSON de la configuration ;
- sauvegarde locale dans le navigateur.

Réglage de départ conseillé pour Lemegeton : **8 frames à 8 FPS**.
