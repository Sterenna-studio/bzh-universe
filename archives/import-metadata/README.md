# Metadonnees d'import

Ce dossier conserve les fichiers systeme arrives avec les lots bruts, lorsque leur presence doit rester auditable sans les melanger aux medias finaux.

Regle appliquee :
- `imports/` reste un sas visible et non ignore ;
- les fichiers sans valeur media ou documentaire directe sont deplaces ici apres inventaire ;
- le chemin sous `archives/import-metadata/<date>/imports/...` conserve leur provenance exacte.
